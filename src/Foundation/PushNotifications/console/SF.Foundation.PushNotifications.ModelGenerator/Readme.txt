# Model Config Files

This project is used to generate & store config files that need to be deployed to xConnect and related services to use UserSettings

https://doc.sitecore.com/developers/93/sitecore-experience-platform/en/deploy-a-custom-model.html

PushNotifications.Model.json is the output of this console app.
sc.PushNotifications.Model was manual configured to be deployed to the marketing automation engine.

Don't forget to deploy the SF.Foundation.PushNotifications dll as well to each service.
